### Bauteile

- DOIT ESP32 DEVKIT V1
- BME280 I2C
- OLED Display I2C
- Light dependent resistor
- Taster
- 2x 10k Ohm Widerstand
- Optional: WS2812B Addressable RGB LED Strip
